所以才去第二种方法

这里的最主要的就是想表达，注册的时候，你输入的是大写的ADMIN也会转换成小写的admin来识别

并且，你在ADMIN账户里改密码时，相当于在给admin的账户更改密码

所以我们要找到第一次接触该函数会变成大写，而第二次则会变成小写的一些字母

第一次识别是大写，第二次是小写的神奇字母

ᴬᴰᴹᴵᴺ

这就是我去网上找的，嘻嘻

链接：https://symbl.cc/en/1D2E/

之后就是登录，在再更改密码

更改密码的时候，我们就把admin也一起改了

所以登陆就行了
